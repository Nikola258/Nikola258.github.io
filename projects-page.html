Tijdens mijn opleiding heb ik ervaring opgedaan met Java.

Een van mijn projecten heet Mastermind en is een applicatiespel.

Het doel van het spel is om een geheime code te raden bestaand uit vier cijfers. Als speler moet je vier cijfers invoeren in de command line om de code te ontcijferen. Je kunt cijfers kiezen uit het bereik van 1-6, waarbij elk cijfer uniek moet zijn, aangezien de geheime code nooit meer dan één exemplaar van elk cijfer bevat. Bovendien heb je slechts 10 pogingen om de code correct te raden.
